📌 Remark:
- While making your app run you can see some error:
npm error Missing script: "dev"
- At that time:
1) Open server folder => package.json file
2) Do following steps:
- Install nodemon
npm install nodemon --save-dev
- In package.json you will see:
"scripts": {
  "test": "echo \"Error: no test specified\" && exit 1"
}
👆 In this scripts section add the following:
"scripts": {
  "dev": "nodemon index.js",
  "start": "node index.js"
}
✔ Finally, it should look like this:
"scripts": {
    "dev": "nodemon index.js",
    "start": "node index.js",
    "test": "echo \"Error: no test specified\" && exit 1"
}
