1) Create Project:
npx create-next-app@latest next-todo
cd next-todo
npm run dev
- First make sure project runs
Open: http://localhost:3000

2) Create Folder Structure:
app
    (public)
        page.tsx
        layout.tsx
        todos
            page.tsx
    components
        Navbar.tsx
    redux
        store.ts
        Provider.tsx
        apis
          todo.api.ts
    types
        Note.ts
db.json

3) Installing Packages:
a) Install Redux:
npm install @reduxjs/toolkit react-redux
@reduxjs/toolkit → easier Redux setup
react-redux → connect Redux with React
b) Install React Hook Form
npm install react-hook-form
- For handling forms
c) Install Zod
npm install zod
- For validation
d) Zod Resolver
npm install @hookform/resolvers
- Connects Zod with React Hook Form
e) In one line:
npm install @reduxjs/toolkit react-redux react-hook-form zod @hookform/resolvers
